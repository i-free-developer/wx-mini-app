1 Get wechat a mini APP ID (optional)

  1. get a mini APP ID at https://mp.weixin.qq.com

    with a mini APP ID, you can submit your code to Tencent and get approved and finally get the mini app released, every one can use it from wechat

  2. download the Integrated Development Environment (IDE) Wechat IDE
  3. ready, go ->

2 Start the mini app project

  1. create a folder some where, such as myminiapp
  2. create a mini app Project from IDE, choose the created folder
  3. input the mini app ID you got in previous step, you can use test model if you don't got a mini APP ID

3 File and code structure

  1. in the project folder, you should have the following folders and files
    • images folder - all images should be here, such as logo, other images in content should be in cloud
    • pages folder - all mini app pages are here
    • app.js - code will be executed when a user enter the app, put some basic business logic and global data here, such as user login
    • app.wxss - css for all pages
    • app.json - configuration of the mini app
    • project.config.json - configuration of the whole project

4 Add some conten and WXML markup

  1. To save time, just right-click the following image and save it inside your WeChat Mini Program code directory (put this file inside the "images" folder)

  2. Then start writing some WXML code in index.wxml. We want:
    • One image tag
    • Two text tags
    • One button tag
  3. Have a little doubt here? no worries, always refer to the Official Documentation!
  4. Your code will look like this:
    5. 
              <image src="/img/kungfupanda.jpg"/>
              <text>Get new superpowers</text>
              <text>Learn to code!</text>
              <button type="default">Start today</button>

Tips

  1. Use the built-in auto complete function to make sure you don't forget any end tags
  2. Your image src allows to load the file from the current page (index.wxml), you want to start with a slash / to write an absolute path
  3. Don't forget to place your images inside a folder, e.g /images
  4. the text tag is "inline", i.e each tag will be side by side. If you want to break on two or multiple lines, consider using view tags instead

5 Add WXSS for Fonts and Colors

  1. First, transform your image tag into a background banner
  2. Second, add CSS classes in your index.wxml. Your code will look like this: 
    3. 
              <view class="banner">
                <view class="title">Get new superpowers</view>
                <view class="proposition">Learn to code!</view>
                <button type="default">Start today</button>
              </view>
  3. Then copy/paste the CSS code below in index.wxss and customize it
    4. 
          .banner {
            color: white;
            text-align: center;
            height: 100vh;
            background-size: cover !important;
